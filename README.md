## Know About Me

Hey there! I'm **Ares** 👋

I build bots, tools, and small web apps mostly out of curiosity and a refusal to do repetitive things by hand. Python is my daily driver, Discord/Telegram bots are my playground, and dark-mode everything is non-negotiable. When something can be automated, it *will* be automated — my sleep schedule, unfortunately, is not one of those things.

I like clean code more than I write it, and I learn best by shipping something small, breaking it, and fixing it at 2 AM.
